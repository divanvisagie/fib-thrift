# fib-thrift


[![Build Status](https://travis-ci.org/divanvisagie/fib-thrift.svg?branch=master)](https://travis-ci.org/divanvisagie/fib-thrift)


Fibonacci over thrift in scala using Finatra
Testing

## Scala Server and Client
`sbt test`

Running Server:

```sh
sbt server/run
```

Running Client:
```sh
sbt client/run
```
